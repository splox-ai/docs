---
title: 'List User MCP Connections'
openapi: 'POST /mcp-tools/list-user-connections'
---

Lists the authenticated user's MCP connections grouped by MCP URL, including available tools and connection IDs to use for execution.

## Usage

<CodeGroup>

```bash cURL
curl -X POST https://app.splox.io/api/v1/mcp-tools/list-user-connections \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{}'
```

```python Python
from splox import SploxClient

client = SploxClient(api_key="YOUR_API_KEY")
resp = client.mcp.list_user_connections()
for group in resp.connections:
    print(group.mcp_url, [c.mcp_server_id for c in group.connections])
```

```typescript Node.js
import Splox from "splox";

const client = new Splox("YOUR_API_KEY");
const resp = await client.mcp.listUserConnections();
for (const group of resp.connections) {
  console.log(group.mcp_url, group.connections.map(c => c.mcp_server_id));
}
```

```go Go
resp, err := client.MCP.ListUserConnections(ctx)
if err != nil {
    panic(err)
}
for _, group := range resp.Connections {
    fmt.Println(group.MCPURL)
}
```

</CodeGroup>
