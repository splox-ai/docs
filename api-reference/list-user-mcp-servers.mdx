---
title: 'List User MCP Servers'
openapi: 'GET /user-mcp-servers'
---

Lists MCP servers configured by the authenticated user.

## Usage

<CodeGroup>

```bash cURL
curl "https://app.splox.io/api/v1/user-mcp-servers" \
  -H "Authorization: Bearer YOUR_TOKEN"
```

```python Python
from splox import SploxClient

client = SploxClient(api_key="YOUR_API_KEY")
resp = client.mcp.list_user_servers()
for server in resp.servers:
    print(server.id, server.name, server.url)
```

```typescript Node.js
import Splox from "splox";

const client = new Splox("YOUR_API_KEY");
const resp = await client.mcp.listUserServers();
for (const server of resp.servers) {
  console.log(server.id, server.name, server.url);
}
```

```go Go
resp, err := client.MCP.ListUserServers(ctx)
if err != nil {
    panic(err)
}
for _, server := range resp.Servers {
    fmt.Println(server.ID, server.Name, server.URL)
}
```
