### Prompt Caching

<Card>
  **Optimize costs with prompt caching:**
  
  - **First request**: Full cost (\$1.25/M for GPT-5)
  - **Cached requests**: 90% discount (\$0.13/M)
  - **Cache duration**: 5 minutes of inactivity
  
  **Perfect for:**
  - Chatbots with large system prompts
  - Document analysis with consistent context
  - Multi-turn conversations
  - RAG applications with static knowledge
</Card>

### Context Windows

OpenAI models support large context:

- **GPT-4o**: Up to **128K tokens** (~100K words)
- **GPT-5**: Up to **128K tokens** (~100K words)
- **o-series**: Up to **128K tokens** (includes reasoning tokens)
